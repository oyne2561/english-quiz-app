## 例題1

**英文:**
Argo Workflows is an open source container-native workflow engine for orchestrating parallel jobs on Kubernetes.

**文構造:**
- **SVC構造** (Subject + Verb + Complement)
  - **S (Subject)**: Argo Workflows
  - **V (Verb)**: is
  - **C (Complement)**: an open source container-native workflow engine for orchestrating parallel jobs on Kubernetes
    - **名詞句**: an open source container-native workflow engine
    - **前置詞句**: for orchestrating parallel jobs on Kubernetes
      - **動名詞句**: orchestrating parallel jobs
        - **目的語**: parallel jobs
      - **前置詞句**: on Kubernetes

**全文の日本語訳:**
Argo Workflowsは、Kubernetes上で並列ジョブをオーケストレーションするためのオープンソースのコンテナネイティブなワークフローエンジンです。

**部分的な日本語訳:**
- Argo Workflows: Argo Workflows（固有名詞）
- is: ～である
- an open source container-native workflow engine: オープンソースのコンテナネイティブなワークフローエンジン
- for orchestrating parallel jobs: 並列ジョブをオーケストレーションするための
- on Kubernetes: Kubernetes上で

**重要単語の解説:**
- **workflow**: ワークフロー。一連の処理手順やタスクの流れを表す。複合語で、work（作業）+ flow（流れ）から成る。
- **engine**: エンジン。ここでは、ワークフローを実行・管理するシステムを指す。
- **container-native**: コンテナネイティブな。コンテナ技術を前提として設計された、という意味。container（コンテナ）+ native（ネイティブな）から成る複合形容詞。
- **orchestrate**: オーケストレーションする。複数のタスクやジョブを調整・統合して実行すること。orchestra（オーケストラ）から派生した動詞。
- **parallel**: 並列の。同時に実行される、という意味。
- **Kubernetes**: Kubernetes（固有名詞）。コンテナオーケストレーションプラットフォーム。

**文法ポイント:**
- **SVC構造**: 主語 + be動詞 + 補語の構造。主語の性質や状態を説明する。
- **前置詞句による修飾**: "for orchestrating parallel jobs on Kubernetes"は、workflow engineを修飾する前置詞句。
- **動名詞**: "orchestrating"は動名詞で、前置詞"for"の目的語として機能している。

## 例題13: 関係代名詞と条件文を含む複雑な文

**英文:**
When you submit a workflow, Argo Workflows creates Pods that run the containers specified in your workflow definition, which allows you to execute complex multi-step processes.

**文構造:**
- **時間の副詞節** + **主節** + **非制限用法の関係代名詞節**
  - **時間の副詞節**: When you submit a workflow
    - **S**: you
    - **V**: submit
    - **O**: a workflow
  - **主節**: Argo Workflows creates Pods that run the containers specified in your workflow definition
    - **S**: Argo Workflows
    - **V**: creates
    - **O**: Pods that run the containers specified in your workflow definition
      - **関係代名詞節**: that run the containers specified in your workflow definition
        - **S**: that (= Pods)
        - **V**: run
        - **O**: the containers specified in your workflow definition
          - **過去分詞句**: specified in your workflow definition
  - **非制限用法の関係代名詞節**: which allows you to execute complex multi-step processes
    - **関係代名詞**: which (前の文全体を先行詞とする)
    - **V**: allows
    - **O**: you
    - **不定詞句**: to execute complex multi-step processes

**全文の日本語訳:**
ワークフローを送信すると、Argo Workflowsはワークフロー定義で指定されたコンテナを実行するPodを作成します。これにより、複雑なマルチステッププロセスを実行できます。

**部分的な日本語訳:**
- When: ～のとき
- you submit: あなたが送信する
- a workflow: ワークフロー
- Argo Workflows: Argo Workflows
- creates: 作成する
- Pods: Pod（複数形）
- that run: 実行する（関係代名詞）
- the containers: コンテナ
- specified: 指定された
- in your workflow definition: あなたのワークフロー定義で
- which: それは（関係代名詞）
- allows: 可能にする
- you: あなたに
- to execute: 実行すること
- complex multi-step processes: 複雑なマルチステッププロセス

**重要単語の解説:**
- **submit**: 送信する。リクエストやジョブを送信すること。sub-（下へ）+ mit（送る）から成る。
- **create**: 作成する。新しいものを生成すること。
- **specify**: 指定する。明確に示すこと。
- **definition**: 定義。仕様や定義。
- **allow**: 可能にする。何かができるようにすること。
- **execute**: 実行する。プログラムやタスクを実行すること。
- **complex**: 複雑な。多くの要素が絡み合っている、という意味。
- **multi-step**: マルチステップの。複数のステップを含む、という意味。

**文法ポイント:**
- **時間の副詞節**: "When"で始まる節が、主節の動作が起こるタイミングを表している。

**重要単語の解説:**

- **pipeline**: パイプライン。処理の流れ。
- **integrate**: 統合する。複数のものを一つにまとめること。
- **CI/CD**: Continuous Integration/Continuous Deploymentの略。継続的インテグレーション/継続的デプロイメント。
- **enable**: 可能にする。何かができるようにすること。
- **automated**: 自動化された。手動ではなく自動的に行われること。
- **deployment**: デプロイメント。アプリケーションの展開。
- **otherwise**: そうでなければ。別の方法では、という意味。
- **require**: 必要とする。何かが必要であること。
- **manual**: 手動の。人間が行う、という意味。
- **intervention**: 介入。手を加えること。

**文法ポイント:**
- **譲歩の副詞節**: "While"で始まる節が、主節と対比される情報を表している。
- **受動態**: "are used"と"can be integrated"は受動態。
- **等位接続詞**: "and"が複数の要素を結んでいる。
- **非制限用法の関係代名詞**: "which"は非制限用法で、前の文全体を説明している。
- **関係代名詞節**: "that would otherwise require"は関係代名詞節で、"processes"を修飾している。
- **仮定法**: "would require"は仮定法で、仮定の状況を表している。
